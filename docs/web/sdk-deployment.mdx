---
title: Deployment
description: Geocortex Web - Deploy a Geocortex Web Application
---

:::warning
This is a temporary deployment pattern for alpha. This will change in the future.
:::

1. Run `npm build`
1. Open up the application in Geocortex Web Designer (on premise or SaaS, depending on your production enviroment) that you want to overwrite with you custom application and custom code.
1. Type `Ctrl + Shift + E` to open up the development editing interface.
1. Copy and paste the app layout and app config from your local Web SDK application to the Geocortex Web Designer Interface
1. Upload the built production library to the `Custom Code` section of the development editing interface.
