Geocortex Mobile

The Geocortex Mobile SDK can also be used to extend Geocortex Workflow functionality for Geocortex Mobile. Follow the instructions in the quick start project to get setup for Geocortex Mobile + Geocortex Workflow custom development.

#Development Enviroment

Visual Studio 2019 is the only officially supported development enviroment for extending Geocortex Workflow Server.

#SDK Samples

The Geocortex Mobile SDK includes various SDK Samples for development, including ones like this one which demonstrate custom workflow functionality.
